# Security Policy

## Supported Versions
AETHEL prioritizes the security of its stable releases.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | ✅ Yes              |
| 1.5.x   | 🛡️ Security Fixes Only |
| < 1.5   | ❌ No               |

## Reporting a Vulnerability
We take the security of our sovereign infrastructure seriously. If you discover a vulnerability, please follow these steps:
1. **Do not open a public issue.**
2. Email your report to: `security@aethel-project.org` (GPG encrypted preferred).
3. Include a detailed description, proof of concept (PoC), and potential impact.

## Our Commitment
* We will acknowledge receipt within **48 hours**.
* We will provide an estimated timeline for a fix.
* We follow a **90-day disclosure policy** before public release.
